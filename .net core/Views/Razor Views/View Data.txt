ViewData is a dictionary object that is automatically created
 up on receiving a request and will be automatically deleted
  before sending response to the client.

-a key(in dictionary) cannot be duplicate and cannot be null.

It is mainly used to send data from controller to view.

    Controller              View
      Action            Presentation Logic &
      
      ViewData          => ViewData



ViewData is a property of a ControllerBase class and is of ViewDataDictionary type
 ViewDataDictionary is a child of IDictionary<string, object?>
 key is of string and value is of object type


ViewData is a property of Microsoft.AspNetCore.Mvc.Controller class and 
    Microsoft.AspNetCore.Mvc.Razor.RazorPage class.

All the views are derived from RazorPage class.

It is of Microsoft.AspNet.Mvc.ViewFeatures.ViewDataDictionary type.


It is derived from IDictionary<KeyValuePair<string, object>> type.
That means, it acts as a dictionary of key/value pairs.
Key is of string type.
Value if of object type.

[Route("home")]
[Route("/")]
public IActionResult Index()
{
    ViewData["pageTitle"] = "Hi this is title from variable!!!";

            PersonGender = Gender.Female

    ViewData["people"] = people;
    return View();  //Views/Home/Index.cshtml

}
